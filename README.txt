NOME DO PROJETO: Controle de Acervo Bibliotecário
DESCRIÇÃO: A ideia do projeto é o controle de uma biblioteca pessoal.    
EQUIPE: Matheus Bockor
FUNCIONALIDADES IMPLEMENTADAS:
*Cadastro de novos livros;
*Listagem dos livros já cadastrados;
*Excluir um livro cadastrado;
*Editar informações de um livro cadastrado;
*Adicionar foto da capa do livro;
*Todas informação são gravadas e lidas através de um banco de dados.
